# 校园互助学习平台 - 团队分工说明

## 技术栈
- 后端：Java + Spring Boot 2.7 + MyBatis
- 前端：Vue 3 + Element Plus
- 数据库：MySQL
- 接口文档：Swagger (启动后端后访问 http://localhost:8080/swagger-ui/index.html)

## 项目结构总览

```
campus-qa/
├── database/
│   └── schema.sql              # 建库建表（已写好，直接执行）
├── backend/                     # Spring Boot 后端项目
│   ├── pom.xml
│   ├── src/main/resources/
│   │   └── application.yml     # 配置（数据库密码改成你自己的）
│   └── src/main/java/com/campusqa/
│       ├── entity/             # 10张表对应的实体类（已写好）
│       ├── mapper/             # 数据库操作（已写好）
│       ├── service/impl/       # 业务逻辑（已写好）
│       ├── controller/         # 接口（已写好）
│       ├── config/             # 跨域、安全、Swagger配置
│       └── common/             # JWT、统一返回格式
├── frontend/                    # Vue 3 前端项目
│   ├── src/
│   │   ├── api/                # 所有接口请求方法（已写好）
│   │   ├── router/             # 页面路由（已写好）
│   │   └── pages/              # 所有页面（已有基本框架）
└── docs/
    └── 分工说明.md              # 本文档
```

## 🔧 启动方式

### 1. 数据库
执行 database/schema.sql 文件，创建 campus_qa 数据库和10张表。

### 2. 后端
用 IDEA 打开 backend 文件夹，等待 Maven 下载依赖。修改 application.yml 里的数据库密码。运行 CampusQaApplication.java。

### 3. 前端
在 frontend 目录下运行：
```
npm install
npm run dev
```
访问 http://localhost:5173

### 4. 接口文档
后端启动后，访问 http://localhost:8080/swagger-ui/index.html
